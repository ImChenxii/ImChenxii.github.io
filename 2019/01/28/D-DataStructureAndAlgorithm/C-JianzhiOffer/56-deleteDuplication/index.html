剑指Offer题解系列（五十六）：删除链表中重复的结点

题目描述:

在一个排序的链表中，存在重复的结点，请删除该链表中重复的结点，重复的结点不保留，返回链表头指针。 例如，链表1->2->3->3->4->4->5 处理后为 1->2->5

解法一: 从头到尾删除

先建立一个虚拟头结点，防止第一个就是重复元素。特殊情况有:

  1. 下个结点为空或者下下个为空，后面则必然不可能重复，可直接返回结果
  2. 连续多个重复，需要保存下重复值，一直向后断开。

时间复杂度: O(N^2); 辅助空间复杂度: O(1)

解法二: 哈希表
设置一个前后指针，前指针比后指针早移动一次。如果前指针和后指针值相同，在哈希表里就已经存在映射，说明有重复，删除此节点。如果不相同，添加进哈希表内，两个指针同时向前移动一位。

时间复杂度: O(N); 辅助空间复杂度: O(N)
