剑指Offer题解系列（六十七）：剪绳子

题目描述:

给你一根长度为n的绳子，请把绳子剪成整数长的m段（m、n都是整数，n>1并且m>1），每段绳子的长度记为k[0],k[1],…,k[m]。请问k[0]xk[1]x…xk[m]可能的最大乘积是多少？例如，当绳子的长度是8时，我们把它剪成长度分别为2、3、3的三段，此时得到的最大乘积是18。

输入描述:
输入一个数n，意义见题面。（2 <= n <= 60）
输出描述:
输出答案
示例:
输入8 输出18

解法一: 动态规划

动态规划一般是自上而下思考，设长度为n的绳子最大成绩为f(n)，剪下一刀后剩下的两段长度是i和n-i，在这两段基础上还可能继续剪(子问题)，所以：f(n)=max(f(i)×f(n−i))，因此得到了动态规划转移方程。
根据动态转移方程，一步步计算。

时间复杂度: O(N); 辅助空间复杂度: O(N)

解法二: 贪心算法

当n>=5时，3(n-3)>=2(n-2)>n，并且只有当n=5时，等号成立，所以应该把绳子剪成更多段的3，然后剩下都是2.

时间复杂度: ; 辅助空间复杂度:O()
