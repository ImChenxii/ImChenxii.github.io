代码面试指南系列（五）：单调栈结构

题目描述:

给定一个不含有重复值的数组arr，找到每一个i位置左边和右边离i位置最近且值比arr[i]小的位置，返回所有位置相应的信息。-1表示不存在。
如: arr = [3, 4, 1, 5, 6, 2, 7]
返回二维数组表示结果: [[-1, 2], [0, 2], [-1, -1], [2, 5], [3, 5], [2, -1], [5, -1]]

解法一: 暴力遍历

对每一个位置向左遍历，找到满足条件的值，然后向右遍历，找到满足条件的值。

解法二: 单调栈

如果找每个位置对应左右最近且比其小的位置，需要让单调栈从栈顶到栈底是严格递减的；反之，如果找对应左右最近且比其大的位置，需要让单调栈从栈顶到栈底是严格递增的。
对于此问题来说，建立栈顶到栈底的单调递减栈，且每个栈中元素是列表元素的索引，开始遍历列表：

  1. 当栈为空时，直接将元素push到栈底
  2. 当栈不为空时，如果当前位置元素比栈顶元素大，直接push进栈，整体仍然满足单调递减
  3. 当栈不为空时，如果当前位置元素比栈顶元素小，为了保证单调性，pop出栈顶元素，同时将栈顶元素的栈内下面元素是离其左边最近的且比其还小的元素，原因是: 1.单调性保证了比其小;2.其左边比其大的元素都在它本身被添加进来的时候pop出去了，保证了是左边最近。而当前位置的元素是离其右边最近且比其小的元素，原因是: 1. 首先有pop这个操作，代表当前位置元素比其小;2.右边元素只要出现比它小的元素时，它就会被pop出去，其pop出去就是因为这个元素，所以这个元素就是第一个让它pop的即右边第一个比它小的。
  4. 当遍历完成后，栈中元素还有剩余，开始处理这些元素。首先这些元素的右边都没有比其小的元素了，因为如果有的话，会在之前的操作中，被pop出去，不会再最终保留在栈中; 这些元素左边比其小的就是其栈中下一个元素，原因和上面pop元素时思想相同。

进阶问题: 当列表中的元素有重复时，单调栈如何解决？

解法: 重复元素累积压入，根据规律寻找满足条件索引

开始和没有重复元素的相同，当有重复元素压入栈时，会找到单调栈中合适的位置，并将其索引添加到列表中。当pop元素时，如果被pop元素下面的列表只有一个元素，代表没有重复，直接返回第一个元素，如果有多个元素，代表下面那个值有重复，取列表最后一个索引，即最晚加入的那个索引作为其左边最小，右边最小仍然是使其产生pop的元素索引。
因为pop出是一个列表，可能这个列表本身就是多个重复元素，对这些元素，左边最小和右边最小是一样的，具体如上面所述。
