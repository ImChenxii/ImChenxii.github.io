LeetCode 题解系列(416)：分割等和子集

题目描述:

给定一个只包含正整数的非空数组。是否可以将这个数组分割成两个子集，使得两个子集的元素和相等。

注意:

每个数组中的元素不会超过 100
数组的大小不会超过 200
示例 1:

输入: [1, 5, 11, 5]

输出: true

解释: 数组可以分割成 [1, 5, 5] 和 [11].  

示例 2:

输入: [1, 2, 3, 5]

输出: false

解释: 数组不能分割成两个元素和相等的子集.

解法一: 动态规划

  1. 先将所有数字累加，如果累加结果 sum 是奇数，必定不能分割两个等和子序列，返回 False。如果是偶数，那么将 sum/2，代表每个子序列的累加和应该是 sum/2。
  2. 现在问题转换为在数组中是否存在一个子序列使得其累加和为 sum/2，可以使用动态规划来解决。
  3. dp[len][sum/2+1]，dp[i][j]代表是在数组 nums[0..i] 是否存在序列使得累加和等于 j。
  4. base case：dp 矩阵的第一行代表只有一个元素，那么只有 nums[0]位置是 true，其余都是 false。
  5. 一般位置：演化为一个背包问题，是否加入当前元素到子序列中，来构成和 j，如果当前 nums[i]>j，那么必定不能选择，如果 nums[i]=j，那么必定能够组成，可以只选择 nums[i]，如果 nums[i]<j，那么可以选择加入，也可以选择不加入。选择加入时，就要看 dp[i-1][j-nums[i]] 是否能构成，不选择时，就要看 dp[i-1][j] 能够构成，有一个满足条件，就能构成，那么就为 true，否则为 false。

时间复杂度: O(Nsum); 辅助空间复杂度: O(Nsum)
