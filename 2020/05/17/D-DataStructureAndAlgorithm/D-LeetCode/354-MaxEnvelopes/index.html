LeetCode 题解系列(354)：俄罗斯套娃信封问题

题目描述:

给定一些标记了宽度和高度的信封，宽度和高度以整数对形式 (w, h) 出现。当另一个信封的宽度和高度都比这个信封大的时候，这个信封就可以放进另一个信封里，如同俄罗斯套娃一样。

请计算最多能有多少个信封能组成一组“俄罗斯套娃”信封（即可以把一个信封放到另一个信封里面）。

说明:
不允许旋转信封。

示例:

输入: envelopes = [[5,4],[6,4],[6,7],[2,3]]
输出: 3
解释: 最多信封的个数为 3, 组合为: [2,3] => [5,4] => [6,7]。

解法一: 动态规划

  1. 首先将一个信封的高度和宽度封装在一个数据结构 Envelope 中，并按照宽度进行排序，但如果宽度相同的信封间按照高度进行降序排序。
  2. 这样保证了宽度上的有序性，可以从小到大选择，而最多的信封个数就是在高度数组上找到最长的递增子序列，高度上递增保证了嵌套信封高度上后面的比前面的大，宽度排序并且让相同宽度高度最高的在最前面，保证了宽度上面的递增性。
  3. 此时退化成寻找高度数组上的最长自增子序列问题，可以借助二分查找和辅助有效区数组达到 O(NlogN)复杂度。

时间复杂度: O(NlogN); 辅助空间复杂度: O(N)
