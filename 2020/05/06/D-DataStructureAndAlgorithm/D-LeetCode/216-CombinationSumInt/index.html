LeetCode 题解系列(216)：组合总和 III

题目描述:

找出所有相加之和为 n 的 k 个数的组合。组合中只允许含有 1 - 9 的正整数，并且每种组合中不存在重复的数字。

说明：

  • 所有数字都是正整数。
  • 解集不能包含重复的组合。

 

示例:
输入: k = 3, n = 7
输出: [[1,2,4]]

解法一: 递归回溯+剪枝提前终止

仍然是类似组合总和构建递归树，只不过初始候选集变为 1~9，并且有了个数限制 k，相当于数组天然有序，并且满足总和为 n 的时候临时列表中个数恰好为 k

  1. 递归终止条件：1.当前递归累加和为 n，但是只有当前临时数组元素个数恰好为 k 时才会添加进最终数组；2.当前可取数字为 0 个，因为之前已经判断过当前累加和是否等于目标值，所以现在肯定不是目标值，直接返回
  2. 递归树的构建：选择当前候选集可能的数字，并且下一轮选择只能选择当前候选集后面的数字来防止重复添加。
  3. 剪枝：1.当前情况下即使取了最大的 k 个数字还小于目标数字，最大和是使用等差数组 n 项和计算；2.即使取了最小的 k 个数字还大于目标数字；3.因为确保取到k个数字，所以候选集肯定不能比 k-1 还小，相当于提前剪枝，对应遍历过程中当前位置取得值不能过于靠后，导致候选集变小

时间复杂度: O(N!); 辅助空间复杂度: O(N)
