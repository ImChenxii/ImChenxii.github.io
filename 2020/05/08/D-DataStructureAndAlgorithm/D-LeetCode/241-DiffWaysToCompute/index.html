LeetCode 题解系列(241)：为运算表达式设计优先级

题目描述:

给定一个含有数字和运算符的字符串，为表达式添加括号，改变其运算优先级以求出不同的结果。你需要给出所有可能的组合的结果。有效的运算符号包含 +, - 以及 * 。

示例 1:

输入: “2-1-1”
输出: [0, 2]
解释:
((2-1)-1) = 0
(2-(1-1)) = 2
示例 2:

输入: “2*3-4*5”
输出: [-34, -14, -10, -10, 10]
解释:
(2*(3-(4*5))) = -34
((2*3)-(4*5)) = -14
((2*(3-4))*5) = -10
(2*((3-4)*5)) = -10
(((2*3)-4)*5) = 10

解法一: 动态规划

  1. 首先将字符串中的数字和运算符按顺序放入两个列表中，统计数字的个数为 N，然后建立一个动态规划矩阵dp[N][N]，dp[i][j] 是一个列表，存储的是[i..j]之间的数字所有可能的运算结果。
  2. 动态规划矩阵的 base case：对角线位置，因为只有一个数字，所以结果肯定是自身，即 dp[i][i]是只包含numList.get(i)的列表。
  3. 根据区间长度来填充动态规划矩阵，即从 2 遍历到 N，找到开始位置下标 i 从 0 遍历到 N-1，结束位置下标自然根据长度就能确定，为：j = i + n - 1。
  4. 然后将得到的区间 [i..j] 使用中间数 s 分割成两部分：[i..s] 和 [s+1..j]，分割根据动态规划结果可以得到 dp[i][s] 和 dp[s+1][j]，是两部分所有可能的计算结果，然后得到 s 分割位置的运算符，根据所有不同的 s 分割组合结果全部装进一个列表中，最终所有的 s 遍历完成，得到结果就是 dp[i][j] 的所有可能计算结果。
  5. 最终返回 dp[0][N-1]就是最终结果。

时间复杂度: O(N); 辅助空间复杂度: O(N)
