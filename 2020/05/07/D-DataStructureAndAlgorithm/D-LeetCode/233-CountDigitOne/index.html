LeetCode 题解系列(233)：数字 1 的个数

题目描述:

给定一个整数 n，计算所有小于等于 n 的非负整数中数字 1 出现的个数。

示例:

输入: 13
输出: 6
解释: 数字 1 出现在以下数字中: 1, 10, 11, 12, 13 。

解法一: 数学归纳

对于个位：

  • 在个位数上，1会每隔10出现一次，例如1、11、21等等；发现以10为一个阶梯的话，每一个完整的阶梯里面都有一个1，例如数字22，按照10为间隔来分三个阶梯，在完整阶梯0-9，10-19之中都有一个1。
  • 但是19之后有一个不完整的阶梯，需要去判断这个阶梯中会不会出现1，易推断知，如果最后这个露出来的部分小于1，则不可能出现1（这个归纳换做其它数字也成立）。
  • 可以归纳个位上1出现的个数为：n / 10 * 1 + (n % 10 != 0 ? 1 : 0)

对于十位：

  • 十位数上出现1的情况是10-19，依然沿用分析个位数时候的阶梯理论，10-19这组数，每隔100出现一次，这次阶梯是100，例如数字317，分析有阶梯0-99，100-199，200-299三段完整阶梯，每一段阶梯里面都会出现10次1（从10-19），最后分析露出来的那段不完整的阶梯。
  • 考虑如果露出来的数大于19，那么就是10个1，因为10-19肯定会出现；如果小于10，那么肯定不会出现十位数的1；如果在10-19之间的，计算结果应该是k - 10 + 1。
  • 可以归纳十位上1出现的个数为：(n / 100) * 10 + (if(k > 19) 10 else if(k < 10) 0 else k - 10 + 1)

对于百位：

  • 在百位，100-199都会出现百位1，一共出现100次，阶梯间隔为1000，100-199这组数，每隔1000就会出现一次。假设数为2139，跟上述思想一致，先算阶梯数*100，完整阶梯中1在百位出现的个数，即n/1000 *100得到前两个阶梯中1的个数。
  • 再算漏出来的部分139，沿用上述思想，不完整阶梯数k>199，得到100个百位1，100<=k<=199则得到k - 100 + 1个百位1。
  • 可以归纳百位上出现1的个数：(n / 1000) * 100 + (if(k >199) 100 else if(k < 100) 0 else k - 100 + 1)

然后整理归纳，设i为计算1所在的位数，i=1表示计算个位数的1的个数，10表示计算十位数的1的个数等等，可以得到归纳式:

  • k = n % (i * 10)
  • count(i) = (n / (i * 10)) * i + (if(k > i * 2 - 1) i else if(k < i) 0 else k - i + 1)
  • 后半部分的if else可以简化为: min(max((n % (i*10))−i+1,0),i)

时间复杂度: O(N); 辅助空间复杂度: O(1)
