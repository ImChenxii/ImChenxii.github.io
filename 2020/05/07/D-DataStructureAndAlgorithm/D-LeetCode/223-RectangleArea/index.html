LeetCode 题解系列(223)：矩形面积

题目描述:

在二维平面上计算出两个由直线构成的矩形重叠后形成的总面积。

每个矩形由其左下顶点和右上顶点坐标表示，如图所示。

示例:

输入: -3, 0, 3, 4, 0, -1, 9, 2
输出: 45
说明: 假设矩形面积不会超出 int 的范围。

解法一: 总面积-重叠

  1. 先分别计算两个矩形的面积。
  2. 判断两个矩形是否有重叠，即判断一个矩形左右上下是否可能在另一个矩形里面。如果没有重叠，直接返回两个矩形面积和。
  3. 如果有重叠，确定重叠矩形的上、下、左、右边界，然后求得一个重叠的矩形面积。
  4. 结果就是两个矩形面积相加减去重叠部分的面积。

时间复杂度: O(N); 辅助空间复杂度: O(1)
