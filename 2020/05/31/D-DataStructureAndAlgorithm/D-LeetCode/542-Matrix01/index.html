LeetCode 题解系列(542)：01矩阵

题目描述:

给定一个由 0 和 1 组成的矩阵，找出每个元素到最近的 0 的距离。

两个相邻元素间的距离为 1 。

示例 1:
输入:

0 0 0
0 1 0
0 0 0
输出:

0 0 0
0 1 0
0 0 0
示例 2:
输入:

0 0 0
0 1 0
1 1 1
输出:

0 0 0
0 1 0
1 2 1
注意:

给定矩阵的元素个数不超过 10000。
给定矩阵中至少有一个元素是 0。
矩阵中的元素只在四个方向上相邻: 上、下、左、右。

解法一: BFS

  1. 对应原始矩阵中是 0 的点，没有影响，对于原始矩阵中是 1 的点，开始 BFS 遍历。
  2. 每次都将四周的点添加到队列中，直到队列弹出的位置元素是 0 时，代表这层遍历是最近的，返回此时的队列清空次数。

时间复杂度: O(N); 辅助空间复杂度: O(N)

解法二：动态规划

  1. 直接在原始矩阵上操作，首先先将 1 的位置变成一个比较大的正数，代表不可能的距离。
  2. 然后开始从左上角遍历矩阵，每次与 0 的最近距离取 dp[i-1][j]+1 和 dp[i][j-1]+1 和 dp[i][j] 里面较小者，代表只和左上角的 0 有关。
  3. 然后开始从右下角遍历矩阵，每次与 0 的最近距离取 dp[i+1][j]+1 和 dp[i][j+1]+1 和 dp[i][j] 里面较小者，代表只和右下角的 0 有关。
  4. 最终结果就是既满足左上部分，又满足右下部分的结果。

时间复杂度: O(N^2); 辅助空间复杂度: O(1)
