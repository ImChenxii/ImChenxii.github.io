LeetCode 题解系列(329)：矩阵中的最长递增路径

题目描述:

给定一个整数矩阵，找出最长递增路径的长度。

对于每个单元格，你可以往上，下，左，右四个方向移动。 你不能在对角线方向上移动或移动到边界外（即不允许环绕）。

示例 1:

输入: nums =
[
[9,9,4],
[6,6,8],
[2,1,1]
]
输出: 4
解释: 最长递增路径为 [1, 2, 6, 9]。
示例 2:

输入: nums =
[
[3,4,5],
[3,2,6],
[2,2,1]
]
输出: 4
解释: 最长递增路径是 [3, 4, 5, 6]。注意不允许在对角线方向上移动。

解法一: 带记忆的 DFS

  1. 建立辅助矩阵 paths，paths[i][j]=true 代表这个点已经走到过，否则可以继续走进去。
  2. 建立辅助矩阵 cache，cache[i][j] 代表这个点出发能走的最长递增路径。
  3. 从(0,0)开始遍历，借助辅助矩阵 paths 进行 DFS 搜索，每搜索到一个新的地方，更新一个地方的 cache。
  4. 后面的地方当搜索到 cache[i][j] 不是 0 的时候，可以直接返回。
  5. 因为是单调递增的，不会出现重复的路径导致错误。

时间复杂度: O(N); 辅助空间复杂度: O(N)

解法二：BFS剥洋葱法

  1. 建立每个点的出度表，出度就相当于四周比自己小的元素个数，代表可以由这个点指向自己。由于边界缺少方向，所以需要补充 0 在四周。
  2. 建立好后开始 BFS 遍历，初始队列中就只有出度为 0 的点，代表最小的位置，代表洋葱的最外层。
  3. 出队时，将其周围比自己大的元素的出度-1，如果周围元素出度为 0 了，代表下一层的点。
  4. 直到最内层，代表的就是最长的递增路径。
