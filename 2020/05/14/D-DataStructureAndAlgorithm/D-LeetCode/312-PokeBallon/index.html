LeetCode 题解系列(312)：戳气球

题目描述:

有 n 个气球，编号为0 到 n-1，每个气球上都标有一个数字，这些数字存在数组 nums 中。

现在要求你戳破所有的气球。每当你戳破一个气球 i 时，你可以获得 nums[left] * nums[i] * nums[right] 个硬币。 这里的 left 和 right 代表和 i 相邻的两个气球的序号。注意当你戳破了气球 i 后，气球 left 和气球 right 就变成了相邻的气球。

求所能获得硬币的最大数量。

说明:

你可以假设 nums[-1] = nums[n] = 1，但注意它们不是真实存在的所以并不能被戳破。
0 ≤ n ≤ 500, 0 ≤ nums[i] ≤ 100
示例:

输入: [3,1,5,8]
输出: 167
解释: nums = [3,1,5,8] –> [3,5,8] –> [3,8] –> [8] –> []
  coins = 3*1*5 + 3*5*8 + 1*3*8 + 1*8*1 = 167

解法一: 递归

  1. 先建立辅助数组helps，在原来气球分数数组 nums 的基础上，扩建两个边界且值都为 1。
  2. 建立递归函数 process(helps, i, j) 代表按一定顺序打破 (i,j) 之间的所有气球且假设 i-1 和 j+1 位置气球都没有破的最大分数。
  3. 递归函数的终止条件：当 i==j 时，代表只有一种选择，打破 i/j 位置的气球，此时的分数就是 helps[i-1]*helps[i]*helps[i+1]。
  4. 递归函数一般位置：对于当前 i~j，选择一个最后打爆的位置，假设为 m，那么打爆后此次的分数就是helps[i-1]*helps[m]*helps[j+1]，打爆后将分为两部分(边界是一部分)，继续两部分继续向下递归，只不过边界位置改变了。
  5. 最终结果就是 process(1, len)。

时间复杂度: O(N!); 辅助空间复杂度: O(1)

解法二：动态规划

  1. 可以由上边的递归优化成动态规划，建立动态规划矩阵 dp[len+1][len+1]，dp[i][j]代表打爆help中[i..j]的所有气球的最高分数，且假设help[i-1]和help[j+1]一定没被打爆。
  2. 动态规划的 base case 就是递归的终止条件，即区间长度只有 1 的时候，即动态规划矩阵的对角线，此时的 dp[i][i]=help[i-1]*help[i]*help[i+1]。
  3. 动态规划的一般位置：此时在区间 i~j 上选择最后要打破的一个气球，枚举所有可能打破的位置，最大分数就是 i..j 的最大分数。动态转移方程：两个边界：help[i - 1] * help[i] * help[j + 1] + dp[i + 1][j] 和 help[i - 1] * help[j] * help[j + 1] + dp[i][j - 1]，中间位置：dp[i][j] = Math.max(dp[i][j], help[i - 1] * help[m] * help[j + 1] + dp[i][m - 1] + dp[m + 1][j]。
  4. 最终返回的就是 dp[1][len] 的结果。

时间复杂度：O(N^3)；辅助空间复杂度：O(N^2)
