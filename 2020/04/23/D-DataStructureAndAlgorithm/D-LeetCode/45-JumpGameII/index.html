LeetCode 题解系列(45)：跳跃游戏II

题目描述:

给定一个非负整数数组，你最初位于数组的第一个位置。

数组中的每个元素代表你在该位置可以跳跃的最大长度。

你的目标是使用最少的跳跃次数到达数组的最后一个位置。

示例:

输入: [2,3,1,1,4]
输出: 2
解释: 跳到最后一个位置的最小跳跃数是 2。
  从下标为 0 跳到下标为 1 的位置，跳 1 步，然后跳 3 步到达数组的最后一个位置。
说明:

假设你总是可以到达数组的最后一个位置。

解法一: 贪心算法

  1. 每次都标记能到达最远的位置，可以标记当前步数最多能走到的位置和多走一步能走到的位置。
  2. 如果当前位置当前步数能够到达，就可以更新下一次多走一步可以到达的更远位置，即 i+nums[i]。
  3. 如果当前位置是当前步数能够达到的最远位置，要想走到下一个位置，就要多走一步，此时就要更新当前能到的最远位置和步数+1。
  4. 如果某次多走一步能到了末尾，说明找到了最少的步数。
  5. 整体过程贪心地只存储当前步数最远能到达的位置，不存储过程中的位置，因为这些位置必可以到达。

时间复杂度: O(N); 辅助空间复杂度: O(1)
