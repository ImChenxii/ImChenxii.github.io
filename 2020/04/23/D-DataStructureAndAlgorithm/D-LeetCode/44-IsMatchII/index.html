LeetCode 题解系列(44)：通配符匹配

题目描述:

给定一个字符串 (s) 和一个字符模式 (p) ，实现一个支持 ‘?’ 和 ‘*‘ 的通配符匹配。

‘?’ 可以匹配任何单个字符。
‘*‘ 可以匹配任意字符串（包括空字符串）。
两个字符串完全匹配才算匹配成功。

说明:

s 可能为空，且只包含从 a-z 的小写字母。
p 可能为空，且只包含从 a-z 的小写字母，以及字符 ? 和 *。
示例 1:

输入:
s = “aa”
p = “a”
输出: false
解释: “a” 无法匹配 “aa” 整个字符串。
示例 2:

输入:
s = “aa”
p = “*“
输出: true
解释: ‘*‘ 可以匹配任意字符串。
示例 3:

输入:
s = “cb”
p = “?a”
输出: false
解释: ‘?’ 可以匹配 ‘c’, 但第二个 ‘a’ 无法匹配 ‘b’。
示例 4:

输入:
s = “adceb”
p = “*a*b”
输出: true
解释: 第一个 ‘*‘ 可以匹配空字符串, 第二个 ‘*‘ 可以匹配字符串 “dce”.
示例 5:

输入:
s = “acdcb”
p = “a*c?b”
输出: false

解法一: 动态规划

  1. 建立一个动态规划矩阵dp，dp[i][j] 代表 s[0..i-1] 和 p[0..j-1]的匹配情况，i=0 和 j=0 的时候代表空串。
  2. base case：s 的空串只能和 p 的空串或者 p 的连续*代表空串的时候为 true，所以 dp[0][j]=dp[0][j-1] && p[j-1]==’*‘
  3. 一般位置dp[i][j]：如果 s[i-1]==p[j-1] 或者 p[j-1]==’?’时候，代表能够单个字符串匹配成功，此时 dp[i][j]=dp[i-1][j-1]；如果不满足且 p[j-1]位置是*，代表*可以匹配 0 个或多个，即此时 dp[i][j]=dp[i][j-1](匹配成功 0 个)||dp[i-1][j](匹配成功多个，因为 dp[i-1][j] 已经处理过了，比如 abcd,ab*)
  4. 最终返回右下角的元素。

时间复杂度: O(N); 辅助空间复杂度: O(N)
