LeetCode 题解系列(40)：组合总和 II

题目描述:

给定一个数组 candidates 和一个目标数 target ，找出 candidates 中所有可以使数字和为 target 的组合。

candidates 中的每个数字在每个组合中只能使用一次。

说明：

  • 所有数字（包括目标数）都是正整数。
  • 解集不能包含重复的组合。

 

例如:
输入: candidates = [10,1,2,7,6,1,5], target = 8,
所求解集为:
[
[1, 7],
[1, 2, 5],
[2, 6],
[1, 1, 6]
]

解法一: 递归回溯+剪枝

  1. 先对数组进行排序，方便后面的剪枝和提前终止
  2. 递归终止条件：满足了目标和，将当前临时列表元素添加到结果列表。其实隐含了一个终止条件，就是到达末尾，只不过后面的遍历都不会遍历，直接到函数末尾返回了。
  3. 构建递归树，仍然是候选集选择元素思想，但是本轮递归选择的元素不能相同，否则会发生重复，所以在这里要进行剪枝，比如[2,2,3,4]，第一个元素添加时，就不能再添加第 2 个 2 了，否则会和第一个后面的重复，但是选择完第 1 个2 后，第 1 个 2 的下一层仍然是有第 2 个 2 的，即下一个候选集是[2,3,4]。并且由于不能重复选取，所以每次开始位置要从下一个位置开始。总结：本层不能选取重复，下一层可以和上一层元素相同。

时间复杂度: O(N!); 辅助空间复杂度: O(N)
