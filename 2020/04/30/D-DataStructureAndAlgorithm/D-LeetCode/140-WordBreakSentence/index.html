LeetCode 题解系列(140)：单词拆分II

题目描述:

给定一个非空字符串 s 和一个包含非空单词列表的字典 wordDict，在字符串中增加空格来构建一个句子，使得句子中所有的单词都在词典中。返回所有这些可能的句子。

说明：

分隔时可以重复使用字典中的单词。
你可以假设字典中没有重复的单词。
示例 1：

输入:
s = “catsanddog”
wordDict = [“cat”, “cats”, “and”, “sand”, “dog”]
输出:
[
  “cats and dog”,
  “cat sand dog”
]
示例 2：

输入:
s = “pineapplepenapple”
wordDict = [“apple”, “pen”, “applepen”, “pine”, “pineapple”]
输出:
[
  “pine apple pen apple”,
  “pineapple pen apple”,
  “pine applepen apple”
]
解释: 注意你可以重复使用字典中的单词。
示例 3：

输入:
s = “catsandog”
wordDict = [“cats”, “dog”, “sand”, “and”, “cat”]
输出:
[]

解法一: 动态规划

  1. 首先建立动态规划矩阵 dp，dp 中存储的都是列表，dp[i] 代表 s[0..i-1]所有可能的由字典组成的情况。
  2. base case：当 i=0 时，代表空字符串，此时dp[0]列表内只有空字符串，可以由词典组成。
  3. 一般位置i：拆分 s[0..i-1]能否由一个字符串+一个单词组成，对于所有可能的切分点，要保证切分出的字符串对应的 dp 列表不为空，并且剩下的一个单词在字典中存在，那么就可以组成，并且当前位置的组成情况加上前面字符串的组成情况+拆分的单词。
  4. 最后一个位置的结果就是最终结果，

时间复杂度: O(N^2); 辅助空间复杂度: O(N^2)
