LeetCode 题解系列(73)：矩阵置零

题目描述:

给定一个 m x n 的矩阵，如果一个元素为 0，则将其所在行和列的所有元素都设为 0。请使用原地算法。

示例 1:

输入:
[
  [1,1,1],
  [1,0,1],
  [1,1,1]
]
输出:
[
  [1,0,1],
  [0,0,0],
  [1,0,1]
]
示例 2:

输入:
[
  [0,1,2,0],
  [3,4,5,2],
  [1,3,1,5]
]
输出:
[
  [0,0,0,0],
  [0,4,5,0],
  [0,3,1,0]
]

解法一: 设立标志位

  1. 每行每列是否有零，就要设立一个标志位，标记这行全是 0，这列也全是 0。
  2. 但是要求原地处理数组，所以只能将标志设置在数组内部，可以将标志设置在第一行和第一列，如果出现了 0，就把对应行第一列设置为 0，将对应列第一行设置为 0，然后再遍历一遍数组，当前位置只要行首或者列首只要一个为 0，当前位置就是 0。
  3. 但第一行和第一列本身没有处理，可以使用布尔变量来标记第一行和第一列，遍历的时候先判断这两个是否满足。

时间复杂度: O(N); 辅助空间复杂度: O(1)
