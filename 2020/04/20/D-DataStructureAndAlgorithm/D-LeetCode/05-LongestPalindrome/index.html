LeetCode 题解系列(5)： 最长回文字串

题目描述:

给定一个字符串 s，找到 s 中最长的回文子串。

解法一: Manacher 算法

Manacher 是用来解决寻找最长回文字串问题的最优解法，步骤如下：

  1. 对字符串进行扩充，如”abc”，扩充后变为”#a#b#c#”，这样不论字符串长度的奇偶个数，都是奇数个。
  2. 建立一个统计回文半径的辅助数组pArr，记录一个所有回文半径，最右即将到达的位置 pR(初始为-1)， 建立一个上次变更 pR 的索引index(初始为-1)，建立一个记录最大回文半径的变量 max和 max 对应的索引 maxIndex。
  3. 开始在扩充后的字符串上进行遍历，判断当前的 pR 是否包住了当前位置 i(pR > i)：1.如果包住了，找寻关于 pR 对应的 index 对称的位置 pArr 的值，根据 pR 和对称位置可以确定的是以当前位置 i 为中心找回文字串的长度可以取对称位置的回文长度和 pR-i 的一个较小值，因为 pR 范围内是回文的，在 pR 范围内能保证和对称位置有相同的回文结果，但对称位置结果长度+i 可能会超过 pR 边界，超过 pR 的部分就不能保证了。即：如果对称位置的回文半径小于 pR-i，即整体回文字串仍然被包住，并且由于关于 index 对称，不会进一步扩大；如果大于或等于 pR-i，那么至少是 pR-i，可能更长，所以赋值 pR-i然后进一步扩散判断。2.如果没包住，赋值最小长度 1，开始扩散判断。
  4. 如果当前位置的最右部分：i+pArr[i]>pR 了，即 pR 最右即将到达的位置更靠右了，需要更新 pR 和对应的 index。
  5. 并更新当前位置的回文半径是否是最大的回文半径，即更新 max 和 maxIndex。
  6. 根据最大半径位置和最大半径复原最长回文字串。

时间复杂度: O(N); 辅助空间复杂度: O(N)
