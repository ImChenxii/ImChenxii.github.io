LeetCode 题解系列(60)：第K个排列

题目描述:

给出集合 [1,2,3,…,n]，其所有元素共有 n! 种排列。

按大小顺序列出所有排列情况，并一一标记，当 n = 3 时, 所有排列如下：

“123”
“132”
“213”
“231”
“312”
“321”
给定 n 和 k，返回第 k 个排列。

说明：

给定 n 的范围是 [1, 9]。
给定 k 的范围是[1,  n!]。
示例 1:

输入: n = 3, k = 3
输出: “213”
示例 2:

输入: n = 4, k = 9
输出: “2314”

解法一: 顺序拼凑

  1. 先计算出 n 的阶乘，然后从高位到低维来凑第 k 个。
  2. 比如确定最高位时：最高位有 k 中选择，选择完一个数字后，后面的数字有(n-1)!种排列的可能，那么最高位取得数字就是应该使得 tmp 个(n-1)!不超过 k。根据 tmp 和当前数字的使用情况，就能确定当前位置的数字。然后开始递归下一层，继续按照这种规律判断。
  3. 注意的是，因为之前使用的数字之后不能再使用，所以判断第 tmp 大的数字时，要从 1 开始除去之前使用过的数字。

时间复杂度: O(N); 辅助空间复杂度: O(N)
