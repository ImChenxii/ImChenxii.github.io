LeetCode 题解系列(22)： 括号生成

题目描述:

数字 n 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 有效的 括号组合。

 

示例：

输入：n = 3
输出：[
“((()))”,
“(()())”,
“(())()”,
“()(())”,
“()()()”
]

解法一: 递归

  1. 递归就要构建递归树，那么递归过程在当前位置可以添加的就两种：左括号和右括号，但是是有限制的。
  2. 限制：当前左括号数目要小于总括号对数才能添加左括号；当前右括号数目要小于左括号数目才能添加右括号。
  3. 递归结束：当构建完所有位置，添加进结果中。

解法二：动态规划

  1. 构建动态转移方程：dp[i]代表的就是使用 i 对括号能组成的所有组合，那么动态转移方程：dp[i]=”(“+dp[可能的括号对数]+”)”+dp[剩下的括号对数]，可能括号对数+剩下的可能对数=i-1，遍历所有的可能来生成
  2. 动态规划的思想就是先生成一对括号，然后转换为子问题即这个括号外部有几对，内部有几对，外部和内部几对就又可以转换为一个当前的动态规划问题。
  3. 动态规划的 base case 就是当 i=0 时，返回空字符串
