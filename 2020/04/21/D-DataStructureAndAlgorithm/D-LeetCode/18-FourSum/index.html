LeetCode 题解系列(18)： 四数之和

题目描述:

给定一个包含 n 个整数的数组 nums 和一个目标值 target，判断 nums 中是否存在四个元素 a，b，c 和 d ，使得 a + b + c + d 的值与 target 相等？找出所有满足条件且不重复的四元组。
注意：答案中不可以包含重复的四元组。

解法一: 固定双位置+去重+提前终止

  1. 根本还是通过固定位置来退化成 TwoSum 来解决，因为复杂度肯定高于 O(NlogN)，所以可以先对数组进行排序，方便后面的去重
  2. 先固定第一个位置 k，从 0 到 arr.length-4，然后固定下一个位置 i，从 k+1 到 arr.length-3，然后第二个位置后面的有序数组使用双指针进行查找满足条件的四数之和
  3. 去重：在两次固定位置上都不能和之前的固定位置重复，否则会重复添加；在移动双指针过程中，如果满足了四数之和的条件，因为是有序数组，所以可能移动指针连续满足，所以跳过相同值，防止重复添加
  4. 提前终止：在确定了两个固定位置后，可以算出当前的最大/小四数之和，即两个固定位置数字+尾/头部两个数字，如果当前的最大/小四数之和比 target 还小/大，可以直接跳过这种固定位置情况，不需要再移动双指针判断

时间复杂度: O(N); 辅助空间复杂度: O(1)
