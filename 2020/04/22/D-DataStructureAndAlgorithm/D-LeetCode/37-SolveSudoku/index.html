LeetCode 题解系列(37)：解数独

题目描述:

编写一个程序，通过已填充的空格来解决数独问题。

一个数独的解法需遵循如下规则：

数字 1-9 在每一行只能出现一次。
数字 1-9 在每一列只能出现一次。
数字 1-9 在每一个以粗实线分隔的 3x3 宫内只能出现一次。
空白格用 ‘.’ 表示。

Note:

给定的数独序列只包含数字 1-9 和字符 ‘.’ 。
你可以假设给定的数独只有唯一解。
给定数独永远是 9x9 形式的。

解法一: HashMap+递归回溯

  1. 为每行、每列、每个 box 建立一个 map，统计 1~9 数字出现的次数。
  2. 初始时，遍历一遍数独数组，将已经放置好的数字标记在 map 中。
  3. 然后开始从(0,0)位置到(row-1,col-1)递归。
  4. 如果当前位置不为’.’，说明是之前已经放置好的位置，直接到下一个位置；如果是’.’，说明要放置数字。首先从 1~9 里面判断是否可放，可以的话就放置，并进行下一步递归；递归返回时会有一个布尔变量标记是否完成填充，如果是 false，说明没有填充到末尾就开始了递归返回，说明这种解法是错误的，需要回溯，回溯时在 map 分别中-1，并标记此位置为’.’。
  5. 下一步的回溯函数：判断是否到行尾和列尾，如果同时到达，说明成功拼成数独，标记布尔变量为 true，如果到达行尾，进行下一行，否则仅仅是进行下一列。

时间复杂度: O(N!); 辅助空间复杂度: O(N^2)
