LeetCode 题解系列(79)：单词搜索

题目描述:

给定一个二维网格和一个单词，找出该单词是否存在于网格中。

单词必须按照字母顺序，通过相邻的单元格内的字母构成，其中“相邻”单元格是那些水平相邻或垂直相邻的单元格。同一个单元格内的字母不允许被重复使用。

 

示例:

board =
[
[‘A’,’B’,’C’,’E’],
[‘S’,’F’,’C’,’S’],
[‘A’,’D’,’E’,’E’]
]

给定 word = “ABCCED”, 返回 true
给定 word = “SEE”, 返回 true
给定 word = “ABCB”, 返回 false  

提示：

board 和 word 中只包含大写和小写英文字母。
1 <= board.length <= 200
1 <= board[i].length <= 200
1 <= word.length <= 10^3

解法一: 递归回溯

  1. 在所有网格中找到和单词第一个字母相同的位置，然后开始在这个位置递归匹配。
  2. 因为不能重复使用同一个位置的字符，所以需要使用一个辅助数组，标记此位置有没有被使用。
  3. 当匹配到字符时，开始上下左右同时匹配，并且字符串指针也右移一位，直到匹配到末尾，说明成功，返回 true。
  4. 当匹配一个字符时，如果这个位置字符匹配成功，将辅助数组对应位置标 true，代表已经使用，递归完这个位置，回溯为 false，方便后面使用。

时间复杂度: O(N); 辅助空间复杂度: O(N)
