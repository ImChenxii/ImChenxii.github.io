LeetCode 题解系列(89)：格雷编码

题目描述:

格雷编码是一个二进制数字系统，在该系统中，两个连续的数值仅有一个位数的差异。

给定一个代表编码总位数的非负整数 n，打印其格雷编码序列。即使有多个不同答案，你也只需要返回其中一种。

格雷编码序列必须以 0 开头。

 

示例 1:

输入: 2
输出: [0,1,3,2]
解释:
00 - 0
01 - 1
11 - 3
10 - 2

对于给定的 n，其格雷编码序列并不唯一。
例如，[0,2,3,1] 也是一个有效的格雷编码序列。

00 - 0
10 - 2
11 - 3
01 - 1
示例 2:

输入: 0
输出: [0]
解释: 我们定义格雷编码序列必须以 0 开头。
  给定编码总位数为 n 的格雷编码序列，其长度为 2n。当 n = 0 时，长度为 20 = 1。
  因此，当 n = 0 时，其格雷编码序列为 [0]。

解法一: 数学推导

使用镜像反射法由 n 阶格雷码推出 n+1 阶格雷码：

  1. 设 n 阶格雷码的集合为G(n)，给 G(n) 的每个前面加 0，得到 G’(n)。
  2. 将 G’(n) 倒序(镜像)为 R(n)，给 R(n) 的每个元素二进制形式最高位前面加 1，即将上面的 0 变为 1，得到 R’(n)。
  3. G(n+1)就是G’(n) 和R’(n)顺序拼接的结果。
  4. 原理：倒序后未加 1 之前的 R(n)，因为 G’(n)是满足格雷码的，那么 G’(n) 的倒序 R(n) 也是满足格雷码的，只是不是从 1 开始，给 R(n) 所有的添加一个 1之后 R’(n) 仍然满足格雷码条件，并且 G’(n) 和 R’(n)是镜像关系，本来在镜像中心位置没加 1 之前是相同的 0 和 1 个数，加 1 之后就相差了一位，所以在镜像中心满足，在镜像两端也都满足，所以整体满足格雷码条件。

时间复杂度: O(N); 辅助空间复杂度: O(1)
